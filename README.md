# Why did I build it?
I built this project because I always use the Pomodoro technique to study but most of the apps were either paid didn’t work properly in the background or didn’t have a to-do section to store tasks so I thought why not make one on my own? And here it is.

# Short Description
This project is a productivity webpage that combines a Pomodoro timer and a To-Do list in one place...

The theme is especially designed to make the place comfortable and suitable for study....
